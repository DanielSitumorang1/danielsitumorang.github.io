## ✨ Fitur Portfolio

- **Tampilan Elegan:** *Premium dark theme* dengan aksen *teal* yang modern dan profesional.
- **Responsif:** Struktur *layout* yang rapi untuk diakses melalui HP, tablet, maupun layar desktop.
- **Interaktif:** Animasi halus saat di-*scroll* (menggunakan Intersection Observer API) serta navigasi lengket (*sticky navbar*).
- **Akses Mudah:** Struktur folder terkelola dengan baik (`assets`, `css`, `js`) dan kode *vanilla* murni yang mudah dibaca.
